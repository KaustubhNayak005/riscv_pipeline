# Byte Pattern Match Benchmark Results
Generated by running string_match.mem on the custom RISC-V 5-stage pipelined processor.

## Output Log
STR
00000001
DONE

## Performance Metrics
- Total Cycles (C): 926 (0x0000039E)
- Instructions Retired (I): 726 (0x000002D6)
- Stalls (S): 67 (0x00000043)
- Flushes (F): 5 (0x00000005)

## Analysis
A very tight iteration loop. Branch mispredictions (flushes) are minimal (5 flushes) because the loop condition is perfectly predicted by the BHT after the first few iterations. Stalls (67) arise from load-use hazards as the data is loaded from memory and immediately compared.
